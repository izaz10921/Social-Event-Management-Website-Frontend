```javascript

```

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>In this case, 1 is a number and "2" is a string. When using the + operator, JavaScript converts 1 to the string "1" and concatenates it with "2", give result in "12".</i>

</p>
</details>
